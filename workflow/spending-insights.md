Raw numbers don't change behavior — advice does. This feature analyzes spending patterns across months and gives plain-language recommendations: what looks like waste, what's worth keeping, and how this month compares to the same period last year. The goal is actionable insight, not just more charts.

## User Stories

- As a user, I want to know if I'm spending more than usual this month so I can course-correct before the month ends
- As a user, I want to see which categories have grown compared to last year so I can understand long-term drift
- As a user, I want to be told specifically where I might be wasting money so I don't have to figure it out myself
- As a user, I want to know which spending habits are healthy so I feel good about keeping them

## Success

- Insights tab or section generates a monthly analysis comparing: this month vs previous 3 months average, and this month vs same month last year
- Analysis is presented as plain-language advice (not just numbers), e.g. "Your dining out spending is 40% higher than your average — consider cooking at home more this week"
- Highlights both negative patterns (overspending) and positive ones (consistent savings, reduced spending in a category)
- Advice is specific to actual spending categories, not generic
- Analysis is generated by an AI model (Claude) using the expense data as input

### Out of Scope

- Budget targets or spending limits — the app doesn't ask you to set a budget
- Real-time alerts during the month — this is a monthly retrospective
- Investment or savings advice — only covers logged expenses
- Advice on subscription pricing (e.g., "Netflix raised prices") — only analyzes what's already logged

## Plan

### Data Inputs

Firebase Function collects and structures:
- This month's spending by category
- Previous 3 months' spending by category (rolling average)
- Same month last year's spending by category
- Total spend comparisons across all three periods

### AI Analysis

The structured household spending data (both users combined) is sent to Claude API with a prompt asking for:
1. Top 2–3 categories showing unusual increases — with a specific observation and a warm, practical suggestion
2. Top 1–2 categories showing positive trends — with genuine encouragement to maintain
3. One overall spending summary sentence

Tone: warm financial advisor — knowledgeable and direct, but kind. Like a trusted friend who happens to know about money. Not cold and clinical, not cheerleader-y.

Response is plain language, concise, actionable. Generated automatically at end of each month.

### Display

A dedicated section inside the Reports tab. Rendered as a short advisory card — not a chart. Auto-generated at month-end; no manual refresh needed.

### Privacy

Expense data sent to Claude API contains amounts and category names only — no notes, no user emails, no personal identifiers.

## Open Questions

- **Minimum history** — what's the minimum data before insights are useful? Need at least 2 months to compare — should the section be hidden until enough data exists, or show a "not enough data yet" message?
